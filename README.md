# HiveBeat
Privacy focused music app based on YouTube Music's API.


## Features
- [x] Search for music
- [x] Play music
- [x] Display data about albums
- [x] Display data about artists
- [x] Manage the queue
- [ ] Create playlists
- [ ] Add Songs to playlists
